# 六.Video

::: demo

```html
<button onclick="openDevice()">打开视频</button>
<video ref="videoA" width="320" height="240" controls></video>
<script>
  async function openDevice() {
    let stream = await navigator?.mediaDevices?.getUserMedia({
      video: true,
      audio: true,
    })
    try {
      var video = this.$refs.videoA
      video["srcObject"] = stream
      video.play()
    } catch (e) {
      console.log("错误信息：" + e)
    }
  }
</script>
```

:::

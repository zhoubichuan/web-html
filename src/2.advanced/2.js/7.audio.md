# 七.Audio

::: demo

```html
<button onclick="openDevice">打开音频</button>
<audio ref="aideoA" controls></audio>
<script>
  async function openDevice() {
    let stream = await navigator?.mediaDevices?.getUserMedia({
      audio: true,
    })
    try {
      var video = this.$refs.aideoA
      video["srcObject"] = stream
      video.play()
    } catch (e) {
      console.log("错误信息：" + e)
    }
  }
</script>
```

:::
